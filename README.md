
# Анализ продаж сети магазинов Харьков

## О проекте
Учебный проект по анализу данных. Исследуется, почему магазин "Дом торговли" отстаёт от остальных четырёх магазинов сети.

## Данные
- 5 магазинов: Епос, Дом торговли, Победа, Героев Труда, Лоск
- Период: май 2026 (30 дней)
- База данных: MySQL (150 строк)

## Инструменты
- MySQL / phpMyAdmin — анализ данных
- Tableau Public — визуализация

## Выводы
- Дом торговли имеет низкий трафик, выручку и количество транзакций
- Магазин работает 9 часов в день против 10 у лидера
- Расположен напротив центрального рынка
